About This Application

This application is designed to provide advanced text and audio processing tools, including emotion recognition, named entity recognition, text punctuation, and more. Developed by Mohamed BEN ALI, a Data Scientist and Speech Scientist, it leverages state-of-the-art machine learning models to deliver accurate and efficient results.

Technologies Used

  • Angular for the frontend framework
  • FastAPI and Flask for backend APIs
  • Transformers and SpeechBrain for machine learning models
  • Flasgger for API documentation
  • Docker for containerization
